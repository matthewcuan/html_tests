Introduction

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web.

By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

Key Components of HTML Elements

The below code will be used as an example of an HTML element.

<p>Graphic design is my passion!</p>

The main parts of an element are as follows:

  1. The opening tag (<p>): This consists of the name of the element (in this case, p), wrapped in opening and closing angle brackets. This states where the element begins or starts to take effect — in this case where the paragraph begins.
  2. The closing tag (</p>): This is the same as the opening tag, except that it includes a forward slash before the element name. This states where the element ends — in this case where the paragraph ends. Failing to add a closing tag is one of the standard beginner errors and can lead to strange results.
  3. The content (Graphic design is my passion!): This is the content of the element, which in this case, is just text.
  4. The element: The opening tag, the closing tag, and the content together comprise the element.
Intro to CSS

CSS (Cascading Style Sheets) is the code that styles web content.

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements.

For example, this CSS selects paragraph text, setting the color to red:

p { color: red; }

How to apply CSS to an HTML Doc

To make the code work, we still need to apply this CSS (above) to your HTML document. Otherwise, the styling won't change the appearance of the HTML.

  1. Open your index.html file. Paste the following line in the head (between the and tags):
    2. <link href="styles/style.css" rel="stylesheet" />
  2. Save index.html and load it in your browser. You should now see your html page stylized.
Survey Form Certification

FreeCodeCamp's first Certification Project for their Responsive Web Design certification was a survey form. This project focsued on the use of the <form> element. Below are key features of the <form> element.

The <input> element is the most used form element. Depending on the type (as listed below), these <input> elements can have different displays and functions.

Type Description
<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (for selecting one of many choices)
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button
Tribute Page

FreeCodeCamp's second Certification Project for their Responsive Web Design certification was a tribute. This project focsued on the use of the <div> element.

The <div> tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

The <div> tag is easily styled by using the class or id attribute. Any sort of content can be put inside the <div> tag!
